An agentic conversational course recommender built on a Graph-RAG architecture, integrating a knowledge graph with an LLM reasoning layer.

The system uses RDF/SPARQL to represent course relationships and dependencies, allowing the LLM to reason over structured data and deliver personalized course recommendations and intelligent class planning.

**Tech stack:** Graph-RAG, RDF/SPARQL, LLM reasoning
